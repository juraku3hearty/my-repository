## 検討事項・リスク(相談で出た論点)

1. **著作権**: HPBやInstagramの写真・文章を無断で使って「完成品」を送ると著作権侵害リスク。
   「希望がなければ即削除」でも複製した時点で侵害は成立するため、削除では遡って合法にならない。
   → **決定した運用(2026-06-10)**: 仮サイトは以下の3原則で作れば実務上問題なし。
   1. 公開しない: noindex+推測されないURL(またはパスワード付き)で、その店にだけURLを送る
   2. 写真はInstagram公式埋め込みを使う: 規約上認められた方法で、店の実際の写真を合法的に表示できる。
      HPB掲載写真はリクルート側に権利があることが多いのでコピーしない
   3. テキストは事実情報のみ: 店名・住所・営業時間・メニュー名・価格は著作権の対象外。HPB紹介文のコピペは避ける
   契約後に本人提供素材へ差し替え(「契約したらちゃんとした写真に差し替わります」をアップセル材料にする)。
2. **HPB規約**: スクレイピングによるリスト作成は規約違反になり得る。リストアップは手作業 or Googleマップ起点で。
3. **予約導線**: 美容系のHPBロックインの本体は予約システム。HPだけでは弱いので
   LINE公式アカウント予約(無料)・Googleで予約などの連携をセットにする。
4. **MEO**: Googleビジネスプロフィール整備の代行をパッケージに含めると月1,980円の説得力が上がる。
5. **価格整合性**: LPは月5,000円表記。先着30店1,980円は「先着30店舗・創業記念価格」と明記して区別する。

### デザインデータの無断利用(勝手に印刷・他媒体転載)対策 ※2026-06-12追加

1. **規約**: 第5条に第3項を追加済み —「当社制作のデザインデータの利用はLink Hokkaido上に限る。
   印刷・他媒体転載は事前許諾(別途データ提供契約)が必要。無断利用にはデータ利用料を請求できる」
2. **実務の防壁(最重要)**: **元データ・高解像度データは渡さない**。Web掲載は画面表示用の解像度のみ。
   印刷に耐えるデータが手元にない=物理的に勝手に刷れない(刷っても粗くて使い物にならない)
3. **クレジット埋め込み**: イベント告知などの制作画像にはデザイン内に小さく「Link Hokkaido」の
   クレジットを入れる。万一転載されてもLHの宣伝になる(被害を広告に転化)
4. **持ち込みデータは別扱い(2026-06-12本人確認)**: 契約店舗が自分で作ったチラシ・告知データの
   持ち込み掲載は**歓迎・プラン内で対応**。権利も内容責任も先方にあるので、こちらは載せるだけでリスクなし。
   制限がかかるのは「LHが作ったデザイン」を外に持ち出すケースだけ
