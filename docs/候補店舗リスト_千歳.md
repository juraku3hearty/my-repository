## 確定前の手動確認手順
1. Googleマップで店名を検索し、プロフィールの「ウェブサイト」欄がHPB/Instagram/食べログ等のリンクか空欄であることを確認する(独自ドメインなら除外)。
2. 「店名 + 千歳 + 公式」「店名 + .com」等で再検索し、独自HP・系列店(チェーン)の有無を確認する。
3. InstagramプロフィールのリンクとHPBページの店舗情報(住所・電話・スタッフ数)を見て、営業中の個人経営店であることを確認してからリスト確定とする。
