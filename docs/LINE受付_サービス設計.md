# LINE受付ハブ サービス設計(2026-06-13・協議中ドラフト)

LINE公式アカウント(Link Hokkaido)を「相談・依頼の**受付＆振り分けハブ**」にする。
用件ごとにフォーム → 自動で **タグ付け・通知・受付返信** → まゆみ/Claude が対応。
※料金の数字・文言は仮。後から調整できる形で作る(まずは大枠の仕組み)。

## お客さんの分類(タグ)
- **見込み**: 契約前。無料仮ページから入る
- **会員**: ページ管理プラン契約者
- **非会員**: 契約なしで単発依頼の人

## 規約(/terms/ = WP page id=215)に追記が必要
- **制作データ(チラシ等)の著作権・利用権は Link Hokkaido に帰属**
- **顧客による印刷・二次利用は不可**(納品は閲覧用・低解像度)

## ハーネスで作る「仕組み」
1. **タグ**: 見込み / 会員 / 非会員 ＋ 用件別タグ(仮ページ希望・イベント・変更・支払い…)
2. **用件別フォーム**: 必要情報を収集 → `on_submit` で「タグ付け＋通知＋受付返信」
3. **リッチメニュー**: LINE下部のボタンで用件を振り分け(新規 / 会員 / 支払い 等)
4. **会員/非会員の出し分け**: タグで判定し、料金案内・対応を自動で変える
5. **WP連携**: チラシ/イベント → `/event/` へ転記(`scripts/wp_post.mjs` の流れ)
