# Threads投稿雛形集(Link Hokkaido用・2026-06-12作成)

X用の業者雛形と同じ使い方ができる、Threads向けの穴埋め式テンプレート。
`【】`の中を埋めるだけで投稿できる。1日1〜2本、完璧じゃなくていいので続けるのが最優先。

## Threadsの基本ルール(Xとの違い)

- 1投稿500文字まで。ただし**短いほうが伸びる**(2〜5行が目安)
- ハッシュタグ文化が薄い。**タグは0〜1個**(Xみたいに並べない)
- 「宣伝」より「独り言・本音」が伸びるアルゴリズム
- 返信(リプライ)が来たら必ず返す。会話が増えるほど表示される
- 営業色の強い投稿は **5本に1本まで**。残りはゆる投稿で信頼貯金

## A. 事業の裏側・正直系

### A-1: 進捗報告
```
【今日やったこと】を、やっと【完了したこと】まで持っていけた。

【正直な感想・つまずいたこと】

ひとつずつやな。
```
例:
> お店の紹介ページ、やっと3軒目のデモまで持っていけた。
> 写真の色味合わせるのに2時間溶けた。誰も気づかんとこやけど。
> ひとつずつやな。

### A-2: 数字の正直公開
```
Link Hokkaido、はじめて【日数/週数】。
【見た人の数/フォロワー/問い合わせ】は【数字】。

【その数字への正直な気持ち】

でも【続ける理由】。
```
例:
> Link Hokkaido、はじめて1週間。問い合わせは0件。
> まあそうよな、まだ誰も知らんもんな。
> でも先週の自分より確実に進んでる。

### A-3: 決めたこと宣言
```

【ひとことオチ・本音】
```

## 運用メモ

- **書けない日は B-2(天気ネタ)**: 一番ハードルが低い。窓の外見て2行書くだけ
- **文体ルール(2026-06-12本人承認)**: 関西弁は「京都の緩め」。コテコテ(大阪系の畳みかけ)にしない。
  「〜やな」「〜なんよね」「〜やから」程度の柔らかい混ぜ方。しつこくならない量が正解
- 関西弁はまゆみさんの武器なのでそのまま出してOK(「〜やな」「〜やって」)。標準語に直すと顔が消える
- 旧グルメ記事・その写真はThreadsでも使用しない(IG初期投稿9本.mdの厳守ルールと同じ)
- 価格を出すときは必ず「先着30店舗(通常5,000円)」の条件をセットで(景表法ルール)
- 業者さんのX雛形は形式の参考のみ。文章はコピーしない(著作権+トーン統一のため)
