# Threads投稿カレンダー 6/12〜6/25(Link Hokkaido用・完成原稿)

業者さんのX雛形と同じ「日付割・コピペで投稿できる完成原稿」形式。
6/20予告 → 6/21ローンチに向けて温度を上げていく構成。

- 【】は投稿時にその日の本当の数字・事実に差し替える(嘘の数字は書かない)
- 価格に触れる投稿は必ず「先着30店舗(通常5,000円)」の条件をセット
- Xの「AI主婦・平井まゆみ」アカウントとは内容を混ぜない(Threadsは地域とお店の話に徹する)。
  ただし矛盾もさせない(聞かれたら「制作を仕組み化してるから安くできる」でつながる)
- 旧グルメ記事・写真は使用しない(厳守ルール継続)
