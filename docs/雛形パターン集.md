## 5つの型と、うちの商品への対応

### A. 店舗紹介型(= 月1,980円の店舗ページの母型)
**該当**: カフェ / イタリアン / ステーキ / 美容室 / ネイル(LPnails) / エステ1 / 工務店1 / 製造業
**構成**: 全画面写真ヒーロー+即CTA → こだわり(写真+文) → メニュー&料金(写真カード or 表) → GALLERY → VOICE → 店舗情報テーブル+地図+「行き方をGoogleMapで開く」 → CTA
**トンマナ**: 飲食=写真主役・暗背景もアリ / 美容=白ベース+ローズ / ネイル・まつげ=GALLERYが命(Instagram埋め込みで代替)
→ **site/templates/salon/index.html v2 に反映済み**

### B. セールスLP型(= 悩み解決訴求。エステ・整体系の営業時/体験コース推しに)
**該当**: エステ2(本命見本・11p) / 鍼灸整骨院 / 接骨院 / 会計事務所
**構成(エステ2の一本糸)**:
1. ヒーロー(実績コピー「最近、若返ったね」)
2. 体験コース価格(定価打ち消し→割引○%OFF) ← 価格アンカリング
3. こんな悩みありませんか?(チェックリスト)
4. 原因の教育コンテンツ(ご存知ですか?)
5. 比較表(当店 vs 代替手段: 費用/痛み/リスク)
6. 実績数 → 3つのケア/理由
7. お客様の声(※個人の感想注記つき)
8. 「とはいえ不安…」→ 流れSTEP1-5で不安解消
9. FAQ(濃色帯)
10. 価格再掲(クロージング) → サロン情報 → ネット予約/LINE予約のWボタン
→ 整体組(Yuni等)への営業と、**うち自身のpartner LPの改良**にも流用可

### C. 体験申込型・スクール(= ピラティス/教室系)
**該当**: ダンス / ピアノ / 料理教室 / 英語教室 / 学習塾
**構成**: ヒーロー+体験CTA → 体験会案内ボックス(ヒーロー直下) → コース紹介(写真+説明×3) → スクールとは(2カラム) → 参加者の声 → FAQ → 電話+フォームのW問い合わせ帯 → 会社概要+地図
→ **site/demos/armour-pilates.html v2 に反映済み**

### D. イベント予約型(= 新規オープン店の「オープン記念」ページに流用)
**該当**: 工務店2(完成見学会)
**特徴**: 完全予約制イベント+日時 → ご来場者プレゼント(クオカード等) → 「3つの見どころ」 → フォーム/電話のW予約
→ 新規オープン枠の店に「オープン記念◯◯」ページを作るときの型

### E. コーポレート型(= Link Hokkaido自身の運営会社情報ページ用)
**該当**: コーポレートサイト(MATSUIRISE) / 歯科クリニック
**構成**: 理念ヒーロー → Mission → サービス → 会社概要 → 採用情報

## 全テンプレ共通の「おおっ」文法(7箇条)

1. 写真が主役: ファーストビューは全画面写真+ロゴ+即CTAボタン
2. CTAのしつこい繰り返し: ヒーロー直下ボックス+中盤+最後の強色帯。常に「WEB(LINE)+電話」の2択
3. お客様の声(VOICE)とFAQは標準装備(声はデモ段階サンプル明記・公開時に実物へ。捏造禁止)
4. 英字小見出し+日本語見出しのペア(Cormorant Garamond イタリックが上品)
5. セクション背景の交互リズム(白/淡色/写真/強色)
6. 店舗情報テーブル+「行き方をGoogleMapで開く」ボタン
7. スマホ最下部に固定バー(電話/Instagram/LINE予約)

## デモ生成の標準手順(トンマナ合わせ・2026-06-10確定)

1. 店のInstagramプロフィール(投稿グリッド)のスクショをもらう(Claude環境からInstagramは見られないため)
2. フィードの基調色・ロゴ色・文字の雰囲気(シャープ/丸み)を読み取る
3. 雛形の :root のCSS変数を店のトンマナに差し替える(色の差し替えは数分)
4. 事実情報({{}}差し替え)→ WPにパスワード付き設置
