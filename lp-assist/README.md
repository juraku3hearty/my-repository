# 🧩 LP相棒

ブロック式のLP作成ソフト（LPツールズ等）で「何を・どの順で並べる？」「どの色がきれい？」に
迷わないための、私用の相棒ツールです。デザインが苦手でもサクサク構成が決められます。

## できること

入力（商材・ターゲット・ゴール・雰囲気）を選ぶと、こう出ます👇

1. **ブロック構成** … 「上から順にこのブロックを置いてね」と具体的に提案
   - 各ブロックに **目的 / 中に入れるもの / 見出し案 / 文字量の目安 / 推奨画像** 付き
2. **画像生成プロンプト** … 各ブロックの画像を AI画像ツール（Midjourney・DALL·E・各種生成AI等）で
   作るための日本語プロンプトをワンタップでコピー
3. **配色パレット** … 雰囲気に合った「きれいに見える」色を HEX付きで提案（クリックでコピー）
   - 黄金比 **70:25:5**（ベース:メイン:アクセント）の使い方ガイド付き
4. **まとめ** … 全部まとめてコピー → メモやソフトに貼るだけ

## 使い方

1. `index.html` を **ダブルクリックでブラウザで開く**（インストール不要・ネット接続も不要）
2. 入力して「この内容で提案する →」を押す
3. 出てきた構成を見ながら、LPツールズに上から順にブロックを置いていく
4. 画像が必要なところは「プロンプトをコピー」→ お好きなAI画像ツールに貼って生成

> 💡 **横に並べて使うコツ**：ブラウザ窓を細くして LPツールズの横に置いても崩れないよう、
> 狭い幅用のレイアウトにしてあります。左にLPツールズ・右にLP相棒、で並走できます。

## 設計メモ

- **LPツールズに依存しません。** ソフト側のブロック事情を知らなくても使える汎用ガイドです。
  （LPツールズの実際のブロック名に合わせたい場合は、ブロック一覧を教えてもらえれば
  `index.html` の `BLOCK` / `FLOWS` をその呼び名に差し替えられます）
- データは全部ブラウザ内で完結。入力内容はどこにも送信・保存されません。
- AIで提案をさらに磨きたい場合（商材に合わせた見出し文や配色の自動生成など）は、
  Claude API をつないだ拡張も追加できます。必要になったら声かけてください。

## カスタマイズしたいとき

`index.html` 1ファイルで完結しています。よく触る場所👇

- `BLOCK` … 各ブロックの説明・文字量・画像プロンプト
- `FLOWS` … ゴールごとのブロックの並び順
- `PALETTES` … 雰囲気ごとの配色
