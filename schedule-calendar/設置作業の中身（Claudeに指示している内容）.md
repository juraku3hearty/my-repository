# 設置作業の中身（Claudeに指示している内容）

このシステムは、**Claude（Claude Code）にフォルダごと渡すと、設置作業の大半を
自動でやってくれます**（→ 別紙「AIに設置してもらう手順.md」）。

ただ「AIに任せると中で何をしているか分からなくて不安」という方のために、
**Claudeに指示している作業の中身**をここに書き出します。
（＝AIを使わず**人が手作業で設置する場合の手順**でもあります）

---

## A. Claudeが自動でやってくれる部分（コマンド作業）

人が手作業でやるなら何に当たるかも併記します。

| Claudeがやること（コマンド） | 人が手でやるなら | 補足 |
|---|---|---|
| ① clasp（GAS用ツール）を用意 | （不要） | `npx @google/clasp` |
| ② **Googleにログイン** | ブラウザでGoogleにログイン | **ここだけ人が「許可」を押す** |
| ③ スプレッドシート付きスクリプトを新規作成 | 新しいスプレッドシート作成 → 拡張機能→Apps Script を開く | `clasp create-script --type sheets` |
| ④ コードをアップロード | Apps Scriptの中身を消し、`Code.gs` を貼り付けて保存 | `clasp push` |

→ ③④が「コードを所定の場所に置く」作業。Claudeはこれをコマンドで一気にやります。

## B. 人がやる仕上げ（Claudeでも手作業でも共通・3つだけ）

### B-1. 初期セットアップを実行
1. できた「予定表→カレンダー」スプレッドシートを開く
2. メニュー「**予定表→カレンダー → ① 初期セットアップ**」をクリック
3. 初回は権限の承認 → 許可
   → 確認シート・設定タブ・「予定表＿未処理」フォルダ・自動読み取りが自動でできます

### B-2. 「設定」タブを入力（学年・名前・共有先）
1. 「**設定**」タブを開く
2. **B1 に学年の数字**（中学1年なら「**1**」／2年「2」／3年「3」）
   → その学年の予定だけ入る（全校行事・部活など学年指定なしは常に入る）
   ★ **進級したら B1 の数字を変えるだけ**（コードは触らない）
3. **B2 にお子さんの名前**（例：たろう）
   → 「**たろうの予定**」専用カレンダーが自動で作られ、そこに登録（メインと混ざらない）
   ※設置した先生ご自身がオーナー＝最初から見られて編集もできる（お子さんが携帯未所持でも管理可）
4. **B3 に共有したい人のメール**（任意。先生本人は不要。ほかのご家族に見せたい時だけ）

### B-3. APIキーを登録
1. 拡張機能 → Apps Script → ⚙️ プロジェクトの設定 → スクリプトプロパティ
2. 名前：**GEMINI_API_KEY** ／ 値：Gemini APIキー（AIza…）
   ※予定表は**画像/PDFの読み取り**を行うため、無料枠では認識が通らないことがあります。
   　Google AI Studio で**お支払い設定（課金）を有効**にしたキーを推奨（費用は月 数十〜数百円程度）。
3. 保存 → メニュー「**② 接続テスト**」で「接続OK」が出れば完了

---

## C. 毎回の使い方
1. 学校・部活などの**予定表（写真/PDF）**を「**予定表＿未処理**」フォルダに入れる
2. 自動で読み取り → 「**確認**」シートに一覧化（怪しい所は色付き）
3. 色の付いたセルと「**対象学年**」列だけ目視チェック・修正
   - 🟡 AIが2回読んで食い違い ／ 🟧 上位AIと食い違い・矛盾 ／ 🟥 読めなかった（要入力）
4. メニュー「**★ カレンダーに登録**」
   → **お子さん名の専用カレンダー**に登録（時刻あり=時間付き／無し=終日）。**二重登録は自動スキップ**。

---

## C-2. （任意）ほかのご家族にも共有（最初の1回だけ・Googleカレンダーで30秒）
先生ご本人はオーナーなので共有不要（最初から見られる・編集できる）。**他に見せたい人がいる場合だけ**：
1. Googleカレンダーを開く
2. 左の「**○○の予定**」（お子さん名のカレンダー）の「⋮」→「**設定と共有**」
3. 「**特定のユーザーやグループと共有**」→ 相手のメールを追加 → 権限「**予定の変更**」→ 保存
   ※自動共有もできますが Calendar API の追加が要るので、まずは手動共有が簡単・確実。

---

## D. 困ったとき
- 読み取りがよく外れる → 写真が暗い/傾いていると精度が落ちます。明るく・まっすぐ撮り直すと改善。
- 学年が変わった → 「設定」タブ **B1 の数字を変えるだけ**。
- データの扱い → 読み取りのため画像はGoogleのAI（Gemini）へ送られますが、**有料APIのため学習には使われません**。
