# AIに設置してもらう手順（Claude / Gemini など）

この「予定表→Googleカレンダー自動登録」一式を、コマンドを実行できるAI
（Claude Code、Gemini CLI などエージェント型AI）に渡せば、設置作業の
大部分をAIが行えます。人がやるのは「ログインの許可」と「APIキー・学年の入力」だけです。

---

## 前提
- Node.js が入っていること
- ターミナルでコマンドを実行できるAI（**Claude Code** / Gemini CLI 等）
  ※Claude Pro / Max のご契約があれば **Claude Code（CLI）** が使えます。
  　ブラウザ版チャット（Claude.ai アプリ）はコマンド実行ができないため、
  　その場合は「手順を教えてもらいながら自分で打つ」形になります。

---

## AIにやってもらう作業（このフォルダで実行）

### 1. clasp（Google Apps Script のCLI）を用意
```
npx --yes @google/clasp --version
```

### 2. Googleにログイン  ← ここだけ人が「許可」を押す
```
npx --yes @google/clasp login
```
ブラウザが開くので、設置先のGoogleアカウントで「許可」する。

### 3. スプレッドシート付きのスクリプトを新規作成
```
rm -f .clasp.json
npx --yes @google/clasp create-script --type sheets --title "予定表→カレンダー" --rootDir src
```
→ 確認シートになるスプレッドシートと、紐づくスクリプトが自動で作られます。

### 4. コードをアップロード
`Code.gs` を `src/` に置いてから：
```
npx --yes @google/clasp push -f
```

## 人がやる仕上げ（3つだけ）

### A. 初期セットアップを実行
1. 作成された「予定表→カレンダー」スプレッドシートを開く
2. メニュー「**予定表→カレンダー → ① 初期セットアップ**」をクリック
3. 初回は権限の承認 → 許可
   → 確認シート・設定タブ・「予定表＿未処理」フォルダ・自動読み取りが自動でできます

### B. お子さんの学年を入力
1. 「**設定**」タブを開く
2. **B1 に学年の数字**を入れる（中学1年なら「**1**」／2年なら「2」／3年なら「3」）
3. これで、予定表から**その学年の予定だけ**カレンダーに入ります
   （全校行事・部活など学年指定のないものは学年に関係なく入ります）
   ★ 進級したら、毎年この **B1 の数字を変えるだけ**（コードは触らない）

### C. APIキーを登録
1. 拡張機能 → Apps Script → ⚙️ プロジェクトの設定 → スクリプトプロパティ
2. 名前：**GEMINI_API_KEY** ／ 値：自分のGemini APIキー（AIza…）
   ※予定表は**画像/PDFの読み取り**を行うため、無料枠だと認識が通らないことがあります。
   　安定させたい場合は Google AI Studio で**お支払い設定（課金）を有効**にしたキーを推奨。
3. 保存 → メニュー「**② 接続テスト**」で「接続OK」が出れば完了

---

## これで完成・使い方
1. 学校・部活などの**予定表（写真またはPDF）**を「**予定表＿未処理**」フォルダに入れる
2. 自動で読み取り → 「**確認**」シートに一覧化（怪しい所は色付き）
3. 内容を確認・修正（特に色の付いたセル、「**対象学年**」列）
4. メニュー「**★ カレンダーに登録**」をクリック
   → 時刻ありは時間付き予定／時刻なしは終日予定で登録。
   → 同じ予定が二重に入らないよう**重複は自動でスキップ**します。

---

## どのAIでも使えるか？
- **Claude Code（Pro/Maxに付属）** … 可（このまま実行）
- Gemini CLI（エージェント型） … 可（同じコマンドを実行できる）
- ブラウザ版チャット（Claude.ai 等） … コマンド実行不可（人が手順を見ながら打つ形）
