# サロンお知らせ自動化 — readyタグ全自動投稿版

GASプロジェクト「サロンお知らせ自動化」
(scriptId: `16r4QeDqN9g-Ia3Ptqz4F7rgEtpp-xWF2ysZJ_MVuMLdeRKDa-LQNmDm7`) の更新版。

## なぜ「readyタグを付けても動かない」のか（元の挙動）

- `ready` は **Vimeo の動画タグ**。
- 元のコードでは、`ready` タグは管理ページ（Webアプリ `?admin=1`）に
  表示する候補を絞るだけで、**自動投稿はしない**設計だった。
- 投稿は管理ページを開いて手動でボタンを押したときだけ実行される。
- 定期トリガー `processOnlineSalonAutomation()` には動画処理が入っていなかった。

→ つまり「タグを付ける → 何も起きない」が正しい元の挙動。
　手動で投稿したい場合は Webアプリの末尾に `?admin=1` を付けて開く。

## この版での変更

1. **`processReadyVideos()` を新設**
   Vimeoで `ready` タグを付けた未処理動画を自動でCyfonsへ投稿し、
   会員へのLINE一斉配信も自動で送る。
2. **`detectCategory()` を新設**（カテゴリ自動判定。動画タイトルから）
   - タイトルに `自賠責` → `自賠責保険（基礎編）`
   - タイトルに `AI` → `AI活用講座①`
   - タイトルに `本講座` → `本講座`
   - どれも該当しなければ → `その他`
3. `processOnlineSalonAutomation()` の冒頭で `processReadyVideos()` を呼ぶ。
4. 管理ページ `?admin=1` の一覧も最新10本→50本に拡大。

## 使い方（運用フロー）

1. Vimeoで動画をアップ → タイトルにカテゴリのキーワードを含める
   （例:「2026年6月 本講座 第3回」「AI活用講座 第2回」「自賠責 基礎編」）
2. その動画に **タグ `ready`（小文字）** を付ける。
3. 定期トリガーが回ると自動でサイト投稿＋LINE配信。

> カテゴリを確実に振り分けたいときは、タイトルに上記キーワードを入れるのがコツ。
> 入れ忘れると「その他」で投稿される。

## トリガー設定（これが無いと自動で動かない）

GASエディタ左の時計アイコン（トリガー）→「トリガーを追加」：

- 実行する関数: `processOnlineSalonAutomation`
- イベントのソース: 時間主導型
- 間隔: **15分おき**（早く投稿したいなら）または 1時間おき

> 二重投稿は「処理済み」シートで防止済み（投稿した動画IDを記録し、
> 次回以降スキップ）。なので何分おきでも同じ動画が二度投稿されることはない。
