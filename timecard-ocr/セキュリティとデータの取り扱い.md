# セキュリティとデータの取り扱いについて

タイムカードは個人情報を含むため、データがどこに保存され・どこへ送られるかを明記します。

## データの流れ

```
タイムカード写真
   └→ 事務所の Google ドライブ（共有フォルダ）        … 事務所アカウント内に保存
        └→ Google の Gemini API（文字の読み取りのみ）  … 読み取りのため送信
             └→ 確認シート（事務所アカウント内）        … 結果を保存
```

- 保存先（写真・確認シート・給与第一）は **すべて事務所の Google アカウント内** です。
- 外部に出るのは **読み取りのために Gemini API へ送る画像** のみで、送信先は **Google** です。
  他社サーバーや当方（構築側）のサーバーには **一切送られません**。

## Gemini API のデータ利用について（重要）

- 本システムは **有料（お支払い登録済み）の Gemini API** を使います。
- **有料 API は、送信したデータを Google の AI モデルの学習（トレーニング）に利用しません。**
  （無料枠は学習に使われる場合があるため、本システムでは有料を必須としています。）
- 通信は HTTPS で暗号化。データは不正利用防止・品質管理のため短期間ログ保持される
  ことがありますが、学習には使われません。
- ※ 規約は変わり得ます。最新の正確な内容は Google の公式ページでご確認ください。

## アクセス制御（事務所側で設定）

- 共有フォルダ「タイムカード＿未処理」と確認シートは、**「特定のユーザーのみ」** に
  共有設定してください（「リンクを知っている全員」は使わない）。
- API キーは **コードに書かず**、スクリプトプロパティに保存しています（外から見えません）。
  キーは社外に共有しないでください。
- 操作・閲覧できるのは、フォルダ／シートを共有された事務所内の方だけです。

## 引き渡し後について

- 引き渡し時に **当方（構築側）の API キーと自動実行は削除**し、**シートの所有権は
  事務所へ移譲**します。
- そのため、**引き渡し後は当方からデータにアクセスできません**。データは事務所が
  完全に管理します。

## 推奨設定チェックリスト

- [ ] フォルダ・シートの共有は「特定のユーザーのみ」
- [ ] API キーは有料（お支払い登録済み）アカウントで発行
- [ ] API キーを社外・第三者に渡さない
- [ ] 不要になった写真・データは事務所の判断で削除
