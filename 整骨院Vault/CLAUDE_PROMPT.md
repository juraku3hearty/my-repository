# Claude に渡す指示文（コピペ用）

このVaultを丸ごと読ませて作業させるときの定型文。状況に合わせて選んでコピー。

---

## ① LP原稿をまるごと作らせる

```
整骨院Vault フォルダの中身を全部読んで、07_LP/LP原稿.md の構成の型に沿って
LP原稿を書いて。素材は 01〜06 のノートから引いてください。
薬機法・景表法に触れる「治る」「完治」などの断定表現は使わないこと。
書けたら 07_LP/LP原稿.md の「▼ 実際の原稿」以下に追記して。
```

## ② 広告コピーを量産させる

```
整骨院Vault の 02_強み・差別化、03_お客様の声、04_ターゲット を読んで、
キャッチコピーを10案、SNS投稿文を3案つくって。
ターゲットの悩みワードを使って刺さる言葉にして。
できたら 06_広告素材/広告コピー案.md に追記して。
```

## ③ 足りない情報を洗い出させる

```
整骨院Vault を読んで、LPを作るのに情報が足りていない箇所を一覧にして。
特に #要確認 タグと空欄になっているところを教えて。
```

## ④ 競合との差別化ポイントを整理させる

```
整骨院Vault の 02_強み・差別化 と 05_競合リサーチ を読み比べて、
うちが競合より勝てるポイントを3つに絞って、LPでの見せ方も提案して。
```

---

## 使うときの前提

- このVault（`整骨院Vault` フォルダ）が **このリポジトリに入っている**ので、
  Claude Code のセッションを開けばそのまま読めます。
- 情報を更新したら Obsidian で編集 → Git に push（または Claude に「pushして」）すれば、
  次のセッションで最新の素材を読んでくれます。
