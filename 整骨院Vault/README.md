# 整骨院 広告・LP制作 Vault

整骨院の広告とランディングページ（LP）を作るための **素材箱** です。
ここに情報を貯めておけば、Claude が全部読んで広告コピーや LP 原稿を生成します。

## 使い方（3ステップ）

1. **入れる** — 各フォルダのノートに、わかる範囲で情報を書き込む（空欄でもOK、後から足せる）
2. **紐づける** — 関連するノートは `[[ノート名]]` でリンク、`#タグ` で横断できるようにする
3. **作らせる** — Claude に「このVaultぜんぶ読んで○○を作って」と頼む（→ `CLAUDE_PROMPT.md` 参照）

## フォルダ構成

| フォルダ | 中身 |
|---|---|
| `00_INDEX.md` | 全体のハブ。ここから各ノートへリンク |
| `01_基本情報/` | 院名・場所・営業時間・料金など動かない事実 |
| `02_強み・差別化/` | 他院との違い、施術の特徴、こだわり |
| `03_お客様の声/` | 口コミ・症例・ビフォーアフター |
| `04_ターゲット/` | 来てほしい客層と、その人の悩み（ペルソナ） |
| `05_競合リサーチ/` | 近隣の整骨院や競合の広告メモ |
| `06_広告素材/` | 使えるキャッチコピー・広告文の案 |
| `07_LP/` | LP の原稿（作成中・完成） |
| `_templates/` | 新しいノートを足すときの雛形 |

## Obsidian での開き方

Obsidian → 「フォルダをVaultとして開く」→ この `整骨院Vault` フォルダを選ぶだけ。

## Claude に渡すとき

`CLAUDE_PROMPT.md` にそのまま使える指示文を用意しています。コピーして使ってください。
