## 2. 三方四正（その宮を読むとき一緒に見る宮）

ひとつの宮を読むときは、その宮だけでなく **三方四正**（影響し合う4つの宮）を合わせて見る。

- **本宮**：読みたい宮そのもの
- **對宮**：真向かいの宮（最も強く影響）
- **三合**：本宮から数えて時計回り／反時計回りに 4つ目の2宮

對宮のペア（向かい合わせ）:

| 本宮 | 對宮 |
|------|------|
| 命宮 | 遷移宮 |
| 兄弟宮 | 僕役宮 |
| 夫妻宮 | 官禄宮 |
| 子女宮 | 田宅宮 |
| 財帛宮 | 福徳宮 |
| 疾厄宮 | 父母宮 |

> 例：恋愛（夫妻宮）を読むときは、向かいの官禄宮も合わせて見る。

## 3. 空宮（主星のない宮）の読み方

主星が入っていない宮は **「空宮」**。このときは **對宮（真向かい）の主星を借りて** 読む（借星安宮）。

- 例：命宮が空宮なら、向かいの遷移宮の主星を借りる。
- 借りた星の意味は、各基準書のその星の項目を使う。
- 空宮は「自分の色が薄く、環境や相手の影響を受けやすい」と捉えるのが基本トーン（たたき台）。

## 4. 身宮（しんきゅう）

命盤には命宮のほかに **身宮** がある。身宮は「後天的に強まる部分・人生の重心」を表す。
身宮が重なった宮のテーマが、その人の人生で比重を増す。

| 身宮の位置 | 重心になりやすいテーマ |
|------------|------------------------|
| 命宮 | 自分自身・生き方 |
| 夫妻宮 | 配偶者・パートナー |
| 官禄宮 | 仕事・社会的役割 |
| 財帛宮 | お金・財 |
| 遷移宮 | 外の世界・対人 |
| 福徳宮 | 精神・楽しみ・価値観 |

## 5. 明るさ・四化の補正

- **明るさ（廟旺得利平不陷）** … `自分のトリセツ_基準書.md` の「明るさの補正欄」を使う。
- **四化（化祿・化權・化科・化忌）** … 同ファイルの「四化の補正欄」を使う。化忌は特に要点。
- **⚠️ 矛盾したら「輝き優先」** … 同じ宮に主星が複数あって、向きが食い違うとき（例：一方が「貯めるの苦手」、他方が「貯めるの得意」）は、**明るい星（廟＞旺＞得＞利＞平＞不＞陷）を主役**に読み、暗い星は脇の味付けにする。「苦手」と「得意」を並べて書かない（読者が混乱する）。
  - 例：財帛に天同[旺]＋太陰[廟] → 太陰が明るいので「コツコツ貯められる」が主、天同は「気負わない」の味付け。

## 8. このサービスでの読みの基本姿勢（SPEC §5 倫理）

- 断定しない・煽らない・大げさに言わない。
- 優劣や運命の決めつけをしない（「傾向」「出やすい」で語る）。
- 健康（疾厄宮）は体質の傾向であり医療の診断ではない、と必ず添える。
- 同意のない他者を勝手にプロファイルしない（相手の命盤を晒さない）。

### 鑑定文の書き方ルール（自分のトリセツ等の文章を書くとき）

- **「その人になれ」と言わない。** 本人の性質に逆らう助言（例：頼るのが苦手な人に「頼って」、
  考える人に「考えるな」、動き続ける人に「落ち着け」）はしない。それは否定であって鑑定ではない。
- **まず、そのままを肯定する。** 短所も「強みの裏返し」「優しさ・誠実さの表れ」として受けとめる。
- **恐れを名指しして溶かす。** その人が止まってしまう理由（例：「頼ると負けな気がする」）を言葉にし、
  「それは弱さじゃない」とそっとほどく。
- **"できる小さな扉"を出す。** 性質の中でできる、ハードルの低い一歩を1つだけ示す
  （例：「頼って」ではなく「『今これ背負ってる』と一人にだけ伝えておく。助けてもらわなくていい、知ってもらうだけ」）。
- **締めは、そっと背中を押す。** やわらかいが、あいまいに濁さず具体的に言い切る。
- トーン：寄り添い型がベース。ズバッと型・ポップは使わない。

### 一人ひとりを別人に書くための差別化ルール（重要・似すぎ防止）

主星だけで書くと、主星の並びが近い人どうしが似てしまう（致命的）。**主星以外の差を"読み味"として必ず使う**：

- **輔星・煞星（六吉・六煞）** … 同じ主星でも、左輔右弼の人＝「助けに恵まれ人と組む」、擎羊陀羅の人＝「鋭い／粘る」、火星鈴星＝「勢い／静かに熱中」、文昌文曲＝「言葉・表現が達者」。`輔星_煞星_雑曜_基準書.md` を引いて、キャラの違いに変える。
- **四化** … 例：破軍＋化祿＝「実りやすい開拓者」／無印の破軍＝「純粋な突破型」。化忌＝「一点集中・のめり込む」。
- **身宮** … 同じ命宮でも身宮が財帛＝「お金が主戦場」、官禄＝「役割が主戦場」、遷移＝「外の世界が主戦場」で、人生の力点が変わる。
- **明るさ（廟旺〜陷）** … 同じ星でも廟＝強く出る／陷＝控えめ、で濃淡をつける。

**禁止**：専門用語（輔星名・煞星名）を生で並べない。必ず「考えてから動く慎重派／勢いで動く突破型」のような、**読んだ印象が変わる普通の言葉**に翻訳する。命盤データは主星＋輔星＋煞星＋四化＋身宮＋五行局＋明るさを丸ごと渡し、「同じ主星でもこの人固有の差がハッキリ出るように」と接地して書く。

## 9. 検出ルールの発火条件（コードの根拠・過剰発火の防止）

> **大原則**：各セクションの「際立つ一言（◯◯タイプ）」を出す/出さないは、ここに書いた条件だけで決める。
> 基準書に条件が無い判定をコードの勘で足さない（広く出すぎて全員同じになるのを防ぐ）。新ルールは必ずここに一文足してから実装する。

### 矛盾の解き方（優先順位）：**格局 ＞ 輝き優先 ＞ 同輝度統合**

同じ宮で星の向きが食い違うときの解決は、必ずこの順で：

1. **格局があれば、それを"統合の答え"にする**。例：財帛に武曲＋貪狼＝武貪格 → 「堅実」と「浪費」を並べず、武貪格の読み（**晩成の富・出入りが大きい・稼ぐ力×掴む力**）一本で書く。格局が答えを持っている（足し算で書かない）。
2. **格局が無ければ、明るい星を主役**にする（§5 輝き優先）。例：財帛 天同[旺]＋太陰[廟] → 太陰主役で「コツコツ貯める」、天同は味付け。
3. **同居かつ同輝度で格局も無いときだけ、統合読み**にする。例：武曲[平]＋破軍[平]＝「稼ぐ力も使う力も両方強く、出入りが大きい」を一つの性質として書く（矛盾させない）。

### 離郷の定義（理由つき・コードはこの原則を参照する）

「離郷（生まれた場所を離れて伸びる）」が出るのは、**命宮の重心が"自分の中"に据わっていない**人。理由ごとに3条件：

1. **空宮命＝重心が外にある → 離郷で開く。**
   命宮に主役の星が無く、向かいの遷移宮から星を借りて読む（借星安宮・§3）。
   ＝「自分」が生まれつき外（遷移＝外の世界）にある配置。だから慣れた内側にこもるより、
   外の土地・人・環境に出るほど、借りてきた星が本領を発揮して人生がひらく。

2. **七殺/破軍命＝動いて切り拓く核 → 離れて活きる。**
   命の軸が殺破狼の開拓星（格局§3）。現状維持・据え置きより、動いて壊して創り直すことで伸びる質。
   生まれた場所の枠にとどまると力が余る。場所・環境を変えるほど、その開拓力がかみ合う。
   （貪狼は社交・欲望寄りで"場所を動く核"とは違うので、ここには含めない。）

3. **天馬が命宮or遷移宮＝移動の星 → 動くと運がつく。**
   天馬は文字どおり移動・旅・遷移の星。これが自分（命宮）か外の出方（遷移宮）に乗っている人は、
   じっとしているより動くことで縁とチャンスが回る。

**逆に、自分の星が命宮にしっかり座る安定型**（例：紫微・天府・天相・天梁などが命宮にある）**は、
土地に根を張って力を出すタイプ**なので離郷は出さない。

> **一般客での発火率の目安（実測・ランダム3000件）：約44%**（空宮16.7%＋七殺破軍15.2%＋天馬命遷18.1%の和集合）。
> 半分は超えないので文言は「離郷で伸びる」のまま。もし将来50%を超えたら、断定を弱め「外で伸びるタイプ」に和らげる。
