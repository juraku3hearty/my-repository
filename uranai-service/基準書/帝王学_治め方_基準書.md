# 基準書 v0 ― 帝王学「治め方」【たたき台】

> **これは下書き（たたき台）。** 公共の帝王学・統治の知恵（人や自分をどう活かし治めるか）を、
> 紫微斗数の14主星アーキタイプに当てて Claude が下書きした。**最終的にはまゆみさんが自分の言葉に直す**前提。
> しびとくんの解釈マスタは不使用。専門用語は出さず、ふつうの言葉で「効果」を伝える。
>
> **使い方（鑑定の書き方ルール）:** 各セクションは必ず2層で書く。
>   1. **診断（紫微斗数）** … その人がどういう性質か（星の出方）。基準書に接地。
>   2. **治め方（帝王学）** … その性質を「王として」どう活かす／治めるか。性質を変えさせず、置き場所と扱い方で活かす。
>
> 核の思想：**賢い王は、将軍を詩人に変えない。戦う場所に置き、たまに"ありがとう"を褒美のように渡す。**
> ＝性質を矯正せず、その性質が最も活きる「場」と「ひと工夫」を授ける。否定でなく采配。

## 命宮の主星14 ― 性質と「治め方」

| 主星（王のタイプ） | 治め方（賢い王はこう活かす） |
|--------------------|------------------------------|
| **紫微**（帝の器・統率） | 王が全部背負うと国は傾く。**任せて、最後の責任だけ取る**こと。弱さを見せられる王にこそ、家臣は本音で仕える。 |
| **天機**（参謀・知恵） | 考えるのが仕事。だが考えるだけでは戦は始まらない。**「七分の考えで動け」**と自分に命じ、動きながら直す。 |
| **太陽**（君主の徳・施し） | 民を照らす太陽。照らし続ければ燃え尽きる。**「沈む時間」を自分に許す**。休むことを、民への責務と心得る。 |
| **武曲**（将軍・財） | 行動で示す将。長い言葉は要らない。**「ありがとう」の一言を褒美のように使う**と、その忠が金にも縁にも変わる。 |
| **天同**（福・和） | 争いを避け、場を和ませる人。前線へ無理に出さず**和ませ役に置く**。ただし**「自分の願いを一つ言う」**ことだけは課す。 |
| **廉貞**（情熱・規律） | 火を抱く将。火を消そうとせず、**「火を向ける一点」を与える**。情熱は的があってこそ城を落とす。 |
| **天府**（蔵・宰相） | 守りの名人。守りを尊びつつ**「小さな冒険を一つ」**命じる。**「蔵は減らない」と安心させてから**動かす。 |
| **太陰**（月・陰の力） | 夜に満ちる静かな力。表舞台を強いず、**潤す役・育てる役に置く**。その繊細さを国の感受性として使う。 |
| **貪狼**（多才・遊軍） | 八方に才あるが散ると中途半端。全部は禁じず、**「今いちばんの一国を、あと一歩深く攻めよ」**とだけ命じる。 |
| **巨門**（弁・専門の論客） | 鋭い舌は敵も作る。**舌を「斬る刀」でなく「分かり合う鍵」に**。決めつける前に**一問**を作法とする。 |
| **天相**（補佐・名宰相） | 支えるのが得意で流されやすい。**「お前の意見を先に聞く」と席を与え**、支える者の願いも国の決定に入れる。 |
| **天梁**（庇護・長老） | 守る人だが説教になりがち。助けは止めず、**「言葉の前にひと呼吸」を儀礼に**。求められて差し出す庇護は深く届く。 |
| **七殺**（猛将・決断） | 単騎で道を拓き、抱え込む。**「助けは求めずとも、背負う荷を一人にだけ告げよ」**。知られるだけで、孤軍は軍になる。 |
| **破軍**（開拓・破壊と創造） | 止まれぬ先鋒。**「走ってよい。ただし帰る城を一つ残せ」**。帰る場所がある兵は、最も遠くまで攻める。 |

## 四化が付いたときの「治め方」補正

| 四化 | 治め方の効き |
|------|--------------|
| 化祿（縁・実り） | その力は**ご縁とチャンスに変わる**。出し惜しまず使うほど返ってくる、と王に信じさせる。 |
| 化權（権力・実行） | その分野で**主導権を握れる**。ただし力みは禁物。「握る」より「任せて導く」へ。 |
| 化科（名声・知性） | **名と助けが集まる**。評価は実の証として受け取り、おごらず、助けに感謝で返す。 |
| 化忌（執着・深掘り） | その分野に**深くのめり込む**。悩みの種にも、極めれば誰にも負けぬ専門にもなる。一点集中の采配を。 |

## 身宮（人生の重心）の「治め方」

身宮が乗る宮＝その王が**最も力を注ぐべき主戦場**。ここを疎かにせず、ここで采配を間違えないことが、一生の充実を決める。
