# 基準書 v0 ― 活かし方（自分の中の登場人物）【たたき台】

> **これは下書き（たたき台）。** 公共の知恵（人や自分をどう活かすか）を、紫微斗数の14主星に当てて下書きした。
> **最終的にはまゆみさんが自分の言葉に直す**前提。しびとくんの解釈マスタは不使用。
>
> **書き方の核（重要・口調ルール）:**
> - 各セクションは2層。**①診断（紫微斗数）＝どういう性質か → ②活かし方＝その自分をどう扱うと活きるか。**
> - ②は「**自分の中のその部分を、ひとりの“登場人物”として擬人化**」して語る（例：中の「考える人」「守りたがりの優しさ」）。
>   ＝この目線が、この商品の面白さ。
> - **口調は寄り添い型（やさしい）。** 「〜してみて」「〜してあげて」。**王・命令・〜せよ・采配・兵 などの偉そう／怖い言い回しは使わない。**
> - 性質を叱って直さない。**その子（部分）が活きる"接し方"をそっと渡す**。否定でなく、扱い方のコツ。

## 命宮の主星14 ― 性質と「活かし方（中の登場人物の扱い方）」

| 主星 | 中の登場人物と、やさしい扱い方 |
|------|------------------------------|
| **紫微** | 中の「まとめ役」。全部背負おうとするので、「人に任せて、最後だけ見ててあげる」でちょうどいい。弱音を見せても、あなたの価値は下がらない。 |
| **天機** | 中の「考える人」。考えすぎて止まりがち。「ぜんぶわからなくても、だいたい見えたら動いていいよ」と背中を押してあげて。 |
| **太陽** | 中の「照らす人」。人のために動きすぎて消耗する。「沈む時間」も自分に許してあげて。休むのも大事な役目。 |
| **武曲** | 中の「やり遂げる人」。言葉より行動の人。無理に語らせず、「ありがとう」のひと言を“ごほうび”みたいに渡すと、ぐっと動きやすくなる。 |
| **天同** | 中の「和ませる人」。争いを避けて自分を後回しに。「自分の願いを一つだけ言ってみる」を、そっと促してあげて。 |
| **廉貞** | 中の「情熱の人」。火を消そうとしないで、「これに燃えたい」という一点を見つけてあげると、いちばん輝く。 |
| **天府** | 中の「守る人」。安定が好きで一歩が出にくい。「土台は崩れないよ」と安心させてから、小さな冒険を一つ。 |
| **太陰** | 中の「感じる人」。夜にひとりで抱えがち。無理に表へ出さず、静かに育てる・潤す場所でこそ活きる。 |
| **貪狼** | 中の「楽しむ人」。あれこれ手を出して散りがち。「今いちばん好きなものだけ、もう一歩深く」と声をかけて。 |
| **巨門** | 中の「見抜く人」。鋭くて疑いがち。決めつける前に「一つ質問してみる」を合言葉に。鋭さが“分かり合う鍵”に変わる。 |
| **天相** | 中の「支える人」。人に合わせて自分を後回し。「あなたはどうしたい？」と、自分にも先に聞いてあげて。 |
| **天梁** | 中の「守りたがりの優しさ」。世話が説教に聞こえることも。「言葉の前にひと呼吸」。求められてから差し出すと、深く届く。 |
| **七殺** | 中の「一人で背負う人」。頼るのが負けに感じる。頼らなくていいから、「今これ抱えてる」と一人にだけ伝えておくと、少し軽くなる。 |
| **破軍** | 中の「壊して創る人」。止まれない。動いていい、ただ「いつでも帰れる場所」を一つ残しておくと、もっと遠くまで行ける。 |

## 四化が付いたときの「活かし方」補正

| 四化 | やさしい一言 |
|------|--------------|
| 化祿（縁・実り） | その持ち味は**ご縁やチャンスに変わりやすい**。出し惜しまず使うほど、返ってくる。 |
| 化權（実行・主導） | その分野で**力を持ちやすい**。力みすぎず、「握る」より「任せて見守る」と楽。 |
| 化科（名声・知性） | **名前や助けが集まりやすい**。評価は素直に受け取り、助けには感謝で返して。 |
| 化忌（執着・深掘り） | その分野に**深くのめり込む**。悩みにもなるけど、極めれば誰にも負けない強みになる。 |

## 身宮（人生で比重が増す場所）

身宮が乗る宮＝その人が**いちばん力を注ぎたくなる場所**。ここを大切にすると、一生の充実につながる。
