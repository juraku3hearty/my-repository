# 紫微斗数 自動鑑定サービス ― 引き継ぎ書 / SPEC v0

> **これは何の案件か:** 紫微斗数（しいとすう／Zi Wei Dou Shu）の鑑定を **Claude で自動生成し、PDF で納品して売る** サービスを、ゼロから作る案件。オーナーはまゆみさん（非エンジニア。AI で人の作業を楽にするツール作り＝「100チャレンジ」が本業）。別事業「しびとくん秘書」とは **別物** で、しびとくんのコード・解釈マスタは流用しない（権利の整理は §2）。

## 0. 最初に読むあなた（Claude Code）へ ― 絶対ルール

- **日本語で応答。コードのコメントも日本語。**
- 鑑定の解釈は、本プロジェクト同梱の **「基準書」だけ** を根拠にする。ネット検索やモデルの一般知識、他の占術体系の知識で鑑定しない。
- 紫微斗数には複数の **流派** があり、流派ごとに星の配置・解釈が大きく異なる。ネット／記憶から引いた情報は別流派の可能性が高く、本サービスの基準書と混ざると鑑定が崩壊する。星・宮・四化（化禄/化權/化科/化忌）の扱いは必ず基準書からのみ引く。基準書に該当項目が無ければ「この流派では該当データなし」と扱い、勝手に補完しない。
- **しびとくん秘書のコード・Vercel・解釈マスタは流用しない**（権利 NG）。命盤計算は iztro、解釈は本プロジェクトの基準書で、独立に作る。
- **煽らない・大げさに言わない。人を勝手にプロファイルしない**（§5 倫理設計が商品の核）。

## 1. 事業概要

紫微斗数の鑑定を Claude で自動生成し、PDF で納品して売る。命盤（その人の一生のカルテ）を計算し、星と宮の配置を基準書に従って解釈、まゆみさんの言葉のトーンでパーソナライズした鑑定文＋ビジュアルを PDF にする。入口は **ココナラ**（占いの巨大市場・集客と決済込み）。将来は自前フロント（web/LINE）で対話まで全自動化。

**強み:** 原価ほぼゼロ（命盤＝iztro 無料、AI＝1鑑定料）。質が商品なのでモデルは Claude Opus。

## 2. 権利の整理（なぜ自作でクリアか）

- 紫微斗数そのものは四柱推命と同じ **"誰のものでもない伝統知識"** ＝方法は自由に使える。
- アウトなのは **「もらったしびとくんシステムの具体物」だけ** ＝裏の Vercel・解釈マスタの文章丸コピー・もらったコードの転用。
- → **自作で全部クリア:** ①命盤計算＝iztro（OSS・商用OK）②解釈＝自分の言葉で書く「基準書」（公共の星の意味を一つの流派でまとめ、Claude を接地）③AI＝Claude ④裏側＝自前。

## 3. 商品構造（命盤＝必須入口、その上に全部乗る）

**入口（これを頼まないと他オプション不可＝技術的に命盤が要る）**
- 人となり（命盤＝あなたはこういう人）：大人版（自己理解）／子ども版＝取扱説明書（親向け）・子どもに見せられる版（本人向け応援）。

**自分軸（命盤1個）**
- 今年（流年・毎年リピート）／この日どうなのか（商談・面接・試合・受験＝客が日付1個持参、受け身で「どうなる？」）／部門特化（仕事＝官禄／恋愛＝夫妻／お金＝財帛／健康＝疾厄）。

**他者・関係軸（命盤2個＋）**
- 相性・関係性（親子・職場・友人・恋愛）＝二人の"関係"を読む／この日（子の試合・受験）。

**吉日選び（「この日どうなのか」とは別物）**
- 「いつがいい？」＝客は日付未定 → 候補期間をスキャンして "おすすめ日" を能動提案。相性とは別＝"いい日の重なり" を探すだけで関係は読まない。
- 2種：自分だけの吉日（命盤1個＝開業・引っ越し）／二人の吉日（命盤2個＝両方プラスの日の重なり＝入籍・結婚）。

## 5. 倫理設計（差別化の核）

基準＝**「同意があるか」**。
- ①時間が聞ける間柄（旦那・恋人・親子）＝同意あり → フル鑑定（相手の命盤も読んでOK）。
- ②聞けない相手（好きな人・片想い）＝同意なし → 参加型の時刻推定＋"二人の関係"だけ出す（**相手を晒さない**＝相手の命盤は出力しない）。

「勝手に誕生日から性格を暴かれるのは嫌やろ」。煽らない・大げさに言わない＋人を勝手にプロファイルしない＝ethical 占いで頭一つ抜ける。

## 6. 時刻推定（定盤）の仕組み ― 紫微の壁の解決

紫微斗数は生まれ時間（時辰＝2時間枠）が要るが、片想い相手の時間は分からない（誕生日しか分からない）。普通の占い師が"無理"で断る壁を商品に変える：
1. iztro で 12時辰ぶんの命盤を出す
2. 各時辰の **短所・クセ** を抽出（"良くないところ" の方が見分けやすい）
3. **ラベル無し＋ランダム順** で12個提示（順番で当てられない）
4. 客が「これが近い」を選ぶ
5. 裏で時辰マッピング → 本格鑑定
6. **時間は最後まで客に伝えない**（倫理）

恋愛が最も稼げる市場。

## 7. 導線・リピートループ

①自分の人となり（¥2,000）で「おおっ当たる」と感動 → ②「好きな人のも見たい」が必ず出る → ③でも誕生日しか分からない（時間の壁）→ ④時刻推定プレミアム（¥8,000〜）だけが壁を超えられる＝唯一の武器 → ⑤背中を押されて告白 → 付き合う → ⑥本当の時間を聞ける（＝同意あり）→ ⑦「確定版で答え合わせ」と客が戻る → 入籍等の吉日選びが節目ごとにぶら下がる（片想い → 結婚まで伴走）。
