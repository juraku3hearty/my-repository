# uranai-service ― 紫微斗数 自動鑑定サービス

紫微斗数の鑑定を Claude で自動生成し、PDF で納品して売るサービス。
全体像・商品設計・価格・倫理・フェーズ計画は **[SPEC.md](./SPEC.md)** を読む。

> ⚠️ このサービスは「しびとくん秘書」とは別物。リポジトリ直下の しびとくん側コード
> （`gas_main.js` / `api/` / `vercel.json` / `コード.js` 等）は **参照も流用もしない**。
> 占いサービスのコードは、すべてこの `uranai-service/` ディレクトリ内で独立に作る（SPEC §2）。
>
> ※本来は完全に別リポジトリが理想。今は同じリポジトリ内の独立フォルダとして運用し、
> しびとくん側のファイルには一切触れない方針。

## ディレクトリ構成

```
uranai-service/
├── SPEC.md                 引き継ぎ書（まず読む）
├── README.md               これ
├── src/
│   └── generate-chart.js   命盤を計算して出力するスクリプト
└── 基準書/
    └── README.md           ★解釈の心臓部の置き場（中身はオーナーが書く）
```

## セットアップ

```bash
cd uranai-service
npm install        # iztro が入る
```

## 命盤を出す

```bash
# node src/generate-chart.js <陽暦YYYY-M-D> <時刻index 0-12> <性別 男|女> [--json]
node src/generate-chart.js 1990-5-20 7 女          # 人が読めるサマリ
node src/generate-chart.js 1990-5-20 7 女 --json   # 機械処理用JSON
```

時刻index（時辰／2時間枠）:

| index | 時間帯 | index | 時間帯 |
|---|---|---|---|
| 0 | 23:00-01:00（早子）| 7 | 13:00-15:00 |
| 1 | 01:00-03:00 | 8 | 15:00-17:00 |
| 2 | 03:00-05:00 | 9 | 17:00-19:00 |
| 3 | 05:00-07:00 | 10 | 19:00-21:00 |
| 4 | 07:00-09:00 | 11 | 21:00-23:00 |
| 5 | 09:00-11:00 | 12 | 23:00-01:00（晩子）|
| 6 | 11:00-13:00（午）| | |

※生まれ時間が分からない場合の「時刻推定」は Phase 3 の商品で扱う（SPEC §6）。
