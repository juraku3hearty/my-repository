# 背景素材（文字なし）の置き場

ここに「文字なしの背景イラスト」3枚を置くと、ジェネレータ（`src/build-html.js`）が
命盤データ＋基準書テキストを自動で重ねて、帝の書（自分のトリセツ）のHTML/PDFを生成します。

## 必要なファイル（この名前で置く）

| ファイル名 | 用途 | 内容 |
|------------|------|------|
| `cover.png` | 表紙背景 | 夜明けの王宮・朝日（文字なし） |
| `court.png` | 朝廷図背景 | 中央＝濃紺ディスク＋周囲に空の円11個（文字なし） |
| `body.png`  | 本文背景 | アイボリー＋薄い透かし（文字なし） |

## 共通仕様

- サイズ：**縦A4 / 2480×3508px / 300dpi**（A4比 1:1.414）
- カラー：RGB（PDF出力用）
- **文字は入れない**（文字はこちらで自動で乗せます）
- 安全エリア：上下左右 **40〜60mm** は装飾を控えめに（文字が乗る余白）

## ⚠️ 重ね方の前提（オーナー念押し）

- **文字色**：表紙（暗い背景）＝**金／白**、本文（クリーム背景）＝**濃い色（藍）**
- 文字は**安全エリア（余白40〜60mm）の中**に置き、背景の飾りと被らせない
- → 背景は中央に余白を取って作る（円の中・本文中央は無地に）

## court.png（朝廷図）の座標合わせについて

円の位置に文字をピタッと合わせるため、`court.png` が入ったら
`src/build-html.js` の `COURT_COORDS`（各円の中心位置％）を実画像に合わせて調整します。
（円が「中央1＋周囲11」になっていれば、そのまま十二宮に対応できます）

## 使い方

```bash
# 3枚を置いたら：
node src/build-html.js 1983-1-8 2 女     # → web/out/ に HTML を生成
# 生成HTMLを Chrome で開く → ⌘P → PDFに保存（余白なし・背景オン）
```
